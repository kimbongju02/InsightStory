정인호
1. 각 디렉토리 설명
    - Config(설정정보)
    - Controller(컨트롤러)-Get, Post 처리
    - Dto(데이터 세팅, API 요청 및 응답 세팅)
    - Service(컨트롤러 세부 비즈니스 로직 정리)

2. API 연동 후 개발 진행 상황
    ㄱ. 1차적으로 GPT API로 프롬프트를 날려 GPT에게 특정 소설을
       만들어 달라는 요구 비즈니스 로직 구현완료
    ㄴ. 최초 소설에 대한 선택지를 누를 시 다음 이야기가 진행되는 비즈니스 로직 구현완료
    ㄷ. 최종적인 소설의 완결 비즈니스 로직과 분기 이동 기능은 구현하지 못했음.
